## ✅ LÖSUNG: Verwende einen Online-Build-Service

### Option 1: GitHub Codespaces (KOSTENLOS & EINFACHST)

1. **GitHub Account erstellen** (falls noch nicht vorhanden):
   - Gehe zu: https://github.com/signup
   - Erstelle einen kostenlosen Account

2. **Repository erstellen**:
   - Klicke auf "+" oben rechts → "New repository"
   - Name: `minecraft-damage-farm-mod`
   - Klicke "Create repository"

3. **Code hochladen**:
   - Klicke auf "uploading an existing file"
   - Ziehe den kompletten Ordner `C:\Users\v1358\Desktop\minecraft-damage-farm-mod` in den Browser
   - Klicke "Commit changes"

4. **Codespace starten**:
   - Klicke auf "Code" → "Codespaces" → "Create codespace on main"
   - Warte 1-2 Minuten

5. **Mod bauen**:
   - Im Terminal (unten) tippe:
   ```bash
   chmod +x gradlew
   ./gradlew build
   ```
   - Warte 5-10 Minuten

6. **JAR-Datei herunterladen**:
   - Links: Öffne `build` → `libs`
   - Rechtsklick auf `damage-farm-mod-1.0.0.jar`
   - "Download"

---

### Option 2: Replit (NOCH EINFACHER)

1. **Gehe zu**: https://replit.com/
2. **Erstelle Account** (kostenlos)
3. **Klicke**: "Create Repl"
4. **Wähle**: "Import from GitHub"
5. **Oder**: Lade Dateien direkt hoch
6. **Führe aus**: `./gradlew build`
7. **Download**: JAR-Datei aus `build/libs`

---

### Option 3: Gitpod (AUTOMATISCH)

1. **Gehe zu**: https://gitpod.io/
2. **Login mit GitHub**
3. **Lade Code hoch**
4. **Terminal öffnet sich automatisch**
5. **Tippe**: `./gradlew build`
6. **Download**: Fertige JAR
